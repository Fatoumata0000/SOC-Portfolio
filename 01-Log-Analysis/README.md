# Projet 01 – Détection d’une attaque par force brute

## Objectif
Détecter et analyser une attaque de type force brute à partir de journaux (logs).

## Scénario
Plusieurs tentatives de connexion échouées sont détectées sur un serveur.
L’objectif est d’identifier l’attaquant, le compte ciblé et l’impact de l’attaque.
